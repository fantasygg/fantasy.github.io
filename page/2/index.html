Redis从入门到精通(七、Redis Sentinel)

发布时间 : 2019-05-08
发布 : 2019-05-08 分类 : Redis从入门到精通

Redis Sentinel 是 Redis官方提供的高可用解决方案。

Redis本身可以实现主从复制,但是并没有自动切换主备的功能,当出现其中一个主实例宕机的情况,客户端将无法执行写入命令,Sentinel可以在没有人工干预的情况下执行故障转移,不影响正常的写入操作。

Sentinel是一个分布式的系统,多个Sentinel进程之间协调工作,一来保证故障检测的误报率,二来保证了Sentinel本身的可用性。

分类: Redis从入门到精通 标签: #Redis
